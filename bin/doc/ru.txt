Данный бот имеет следующую структуру

Старт
|__ Управлять FAQ
     |__Добавить вопрос
     |__Удалить вопрос
     |__Редактировать вопрос
     |__Очистить FAQ
|__Посмотреть FAQ
|__Управлять вопросами
     |__Очистить вопросы
|__Посмотреть вопросы
     |__Посмотреть статистику вопросов
     |__Посмотреть список вопросов
|__Управлять пользователями
     |__Добавить роль
     |__Изменить роль
     |__Удалить роль
|__Посмотреть пользователей
     |__Посмотреть статистику пользователей
|__Задать вопрос
|__Ответить на вопрос
__SPLIT__
Описание команд:

<b>Добавить вопрос FAQ</b> - позволяет добавить часто задаваемый вопрос. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот.

<b>Удалить вопрос FAQ</b> - позволяет удалить часто задаваемый вопрос. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. (ID вопроса - его номер в списке)

<b>Редактировать вопрос FAQ</b> - позволяет изменить часто задаваемый вопрос. Для этого нужно нажать на команду, выбрать, что изменить (вопрос или ответ), и выполнить последовательность действий, которые присылает бот. (ID вопроса - его номер в списке)

<b>Очистить FAQ</b> - позволяет полностью очистить список FAQ. Для этого нужно нажать на команду и затем на кнопку подтверждения.

<b>Посмотреть FAQ</b> - позволяет открыть список часто задаваемых вопросов. Для этого нужно нажать на команду. Список выводится по группам 9 вопросов, с помощью кнопок <b>«</b> и <b>»</b> можно пролистывать группы вопросов. Для просмотра ответа нужно нажать на кнопку с вопросом.

<b>Очистить вопросы</b> - позволяет очистить весь список вопросов за всё время.

<b>Посмотреть статистику вопросов</b> - позволяет посмотреть статистику по вопросам. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. Статистика включает:
- <i>общее количество вопросов за всё время</i>
- <i>количество не обработанных вопросов</i>
- <i>количество обрабатываемых вопросов</i>
- <i>количество обработанных вопросов</i>
- <i>среднее время ответа на вопрос</i>

<b>Посмотреть список вопросов</b> - позволяет посмотреть n-ое количество последних вопросов. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот.

<b>Добавить роль</b> - позволяет добавить роль* пользователю Telegram. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. Установить роль можно лишь для тех пользователей, роль которых ниже вашей по уровню.

<b>Изменить роль</b> - позволяет изменить роль* пользователю Telegram. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. Изменить роль можно лишь для тех пользователей, роль которых ниже вашей по уровню.

<b>Удалить роль</b> - позволяет удалить роль* у пользователя Telegram. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. Удалить роль можно только у пользователей, роль которых ниже вашей по уровню.

<b>Посмотреть статистику пользователей</b> - позволяет посмотреть статистику по пользователям с ролями*ю Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. Статистика включает:
- <i>username пользователя</i>
- <i>количество обработанных вопросов</i>

<b>Задать вопрос</b> - задать вопрос пользователям с ролями*. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. Ваш вопрос сохраняется на сервере анонимно. Когда пользователь с ролью* ответит на него, ответ придёт в виде реплая на сообщение, в котором он был задан, если вы удалите сообщение с вопросом, то ответ пришлётся в качестве обычного сообщения.

<b>Ответить на вопрос</b> - ответить на вопрос от пользователя без роли*. Для этого нужно нажать на команду и выполнить последовательность действий, которые присылает бот. Ваш ответ будет анонимно отправлен в чат пользователю, который его задал.
__SPLIT__
Роли:

<b>Developer</b> - имеет доступ ко всему функционалу. Уровень 3.

<b>Administrator</b> - имеет доступ ко всему функционалу, кроме <i>Задать вопрос</i>. Уровень 2.

<b>Moderator</b> - имеет доступ ко всему функционалу, кроме <i>Задать вопрос</i>, <i>Управлять FAQ</i>, <i>Управлять вопросами</i>. Уровень 1.

<b>User</b> - имеет доступ только к функционалу <i>Задать вопрос</i>, <i>Посмотреть FAQ</i>. Уровень 0.
